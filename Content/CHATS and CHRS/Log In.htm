Logging In To HC Standard® Hospital Alert Tracking

There are two different ways to launch HC Standard® Hospital Alert Tracking. The login page can be accessed through any web browser, by entering the access URL provided by the HC Standard® Admin. HC Standard® Hospital Alert Tracking can also be accessed when logged into HC Standard® and does not require an additional log in.

Logging In Using the HC Hospital Alert Tracking URL

Logging In Using the HC Hospital Alert Tracking URL

The HC Standard® Hospital Alert Tracking login prompt ensures that only authorized users gain access. Enter the HC Hospital Alert Tracking URL.

Login to HC Hospital Alert Tracking, as shown in FIGURE 1-1:

  1. Select the Login link.
  2. Enter the valid User Name Password provided by an Admin.
  3. Select the SIGN IN button.

Login Dialog

The login may fail for the following reasons:

  • An invalid username/password combination will lead to an error message. Proceeding into the HC Standard® application will not be possible until a valid username and password are entered.
  • Three consecutive attempts to log in with a valid username, but an invalid password will result in an error message stating, Account is Locked Out. Logging into the system with that specific username will not be possible (even with the valid password) for the predetermined lockout period.
 

Note: The default configuration lockout period is 15 minutes. In order to log in within this period, an Admin must be contacted. Admins have access to manually unlock accounts at any time.

To launch HC Standard®, open any web browser and enter the organization's access URL into the navigation bar.

  • HC Standard® URL: ______________________________________
  • MIEMSS HC Standard® URL: https://miemssalert.com

Each organization is provided a unique access URL. If the access URL is unknown, please contact GER Technical Support by calling 1.866.242.4035 or by emailing support@ger911.com.

On the HC Standard® login dialog, as shown in FIGURE 1-2:

  1. Type the valid Username and Password, provided by an Admin.
  2. Select the Sign In button.

Login Dialog

To access the application launcher, as shown in FIGURE 1-3:

  1. Select the HC logo.
