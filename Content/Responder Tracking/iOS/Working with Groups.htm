Working with Groups of Responders

Tracking a group of responders is simple with HC Standard® Responder Tracking.

Group (or Team) is used when all responders being entered into the device have common information. Below are a few examples of scenarios where group would be most effective:

  • All responders are being assigned to the same mission.
  • All responders are checking in at the same time.
  • All responders have the status update.

Team Button

 
  1. Tap on the Team button on the Main Screen, as shown in FIGURE 1-1.

Group FLEO ID List

Build the FLEO list, as shown in FIGURE 1-2:

  1. Manually type the FLEO ID and tap the + button to add the ID to the Group List.

Once all the responders are entered into the list, swipe or tap the tab to enter information for the group. This is performed in the same manner as an Individual responder.

When the user is done entering information for the group of responders, tap the Menu icon and Save.

Group Options

Group Options

Group options include, as shown in FIGURE 1-3:

  1. Find – Allows the user to search for an ID that was entered into the list.
  2. Scan - Allows the user to scan a FLEO ID.
  3. Recent – Allows the user to retrieve a recent group list entered on the device.
  4. Remove All – Removes all FLEO IDs only from the list/group.
  5. Remove – Tap a FLEO ID and then select Remove to remove the ID from the list/group.
