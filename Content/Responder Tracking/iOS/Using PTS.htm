Using Responder Tracking

Getting Started

Launch HC Mobile™

Launch HC Mobile™

  • Tap the HC Mobile icon to launch Responder Tracking.

Login to HC Mobile™

Login to HC Mobile™

Login Screen

Log in to begin adding responder records, as shown in FIGURE 1-1.

  1. Enter Username and Password as provided by an HC Standard® Administrator.
  2. Tap the Login button.

    OR

  3. If username and password are inaccessible, the user can tap the Skip Login button to enter the HC Mobile™ application. Responders can be entered or checked-in to HC Mobile™ under the Responder Tracking module at this point; however, no data will synchronize to the server until proper user credentials are entered.

 

HC Mobile™ Application Launcher

The modules available to the user will display on the user’s device, as shown in FIGURE 1-2.

  1. From the application launcher, select Responder Tracking.

If it is the first time launching the ResponderTracking module and there is a data connection after logging in, the Synchronization screen will appear, as shown in FIGURE 1-3. Once the synchronization process is complete it will display the main screen. If a user wishes to return to the main screen sooner:

Sync Screen
  1. Tap the Tracking icon in the navigation tray to continue.

If there is no data connection or the Responder Tracking module is still open from a previous log in, the user will be directed to the Responder Tracking Main Screen.

Responder Tracking Main Screen

Responder Tracking Main Screen

Main Screen

The Responder Tracking Main Screen, as shown in FIGURE 1-4:

  1. Connection Status - Shows current connection status to the server.
  2. Sync Status - Shows last date and time the device synchronized. FLEOs on device shows how many records are currently on the user's device. A green icon means there are no records and a yellow icon indicates there are still records to be synchronized.
  3. Current Disaster Information - Shows current Disaster settings.
  4. Buttons - Starts responder entry mode.
  5. GPS - Shows current GPS data .
  6. Sync - Controls data synchronization.
  7. About - Displays HC Responder Tracking™ version information.
  8. Summary - Presents memory, battery and GPS status.

If a Disaster ID has not been set, then upon the entry of the first responder, the user will be prompted to enter the appropriate Disaster Prep information. Entering Disaster Prep information will only be necessary the first time using the device for a specific check-in. Once this information has been entered, the user will be able to check-in multiple responders and will not need to go through this step. For more information, See "Disaster Prep Information"
 

Note: The device may occasionally lose its connection with the server. If this occurs, responder records will remain on the device’s internal memory but will not be uploaded to the database. The device should Auto Synchronize with the database once the connection is re-established.

Account Settings

Account Settings

Account Settings

 

Users are able to access account settings from the Menu,as shown in FIGURE 1-7.

  1. Enter or change the EMAIL address for the account.
  2. Enter or change the PHONE NUMBER associated with the account.
  3. Click or tap Change Password to change the password for the account.
  4. Click or tap the <Back button to go back to the previous page.

Account Settings

To change the password, as shown in FIGURE 1-8:

  1. Enter the OLD PASSWORD.
  2. Enter the NEW PASSWORD.
  3. Enter the new password again under CONFIRM PASSWORD.
  4. Click or tap Change Password to save changes.
  5. Click or tap Cancel to abandon changes.

Disaster Prep

Disaster Prep

The Disaster Prep Panel, as shown in FIGURE 1-9, is where the user selects key elements for check-in. This only needs to be done once during a check-in period since the information applies to all responders. Setting the correct Diaster Prep information is a critical step of the HC Responder Tracking™ log in process. Each selection made here impacts every responder record scanned and saved. Selecting the wrong disaster ID or not selecting a disaster id may cause records to be miscategorized from an overall view.

Items are all selected from drop-down lists which the HC Standard® System Administrator has created in HC Standard®.

Disaster Prep Panel
  1. DISASTER ID - Every responder scanned will have this disaster id associated with the record.
  2. Tap Done once the user has made the appropriate selection(s).
 

Note: If a user realizes a mistake was made when choosing the Disaster ID and the user entered and synched FLEO records, notify a system administrator so that any records previously entered may be corrected.
