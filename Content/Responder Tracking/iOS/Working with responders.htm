Entering Responder Information

Entering Responder Information

Once the profile has been set, submitting responder records is simple.

Responder Entry

 
  1. Tap the Individual button on the Main Screen, as shown in FIGURE 1-1.

FLEO ID

The next step requires the user to enter a FLEO ID, as shown in FIGURE 1-2:

  1. Enter a FLEO ID. The FLEO ID a unique string assigned to a responder. All information on the responder is stored under this ID and is available for recall and updating at later dates. The FLEO ID is a combination of agency and credential number.  For example, an ATF responder with credential 4455 should be entered as atf4455.
  2. Tap OK.
 

Note: The FLEO ID is the only required field when recalling a responder record. If a user has device connectivity and enters a FLEO ID that does not show any data, the user should verify they entered the FLEO ID correctly and try again.

Responder Tracking Panels

Responder Tracking Panels

Once a FLEO ID is entered, begin updating responder information by tapping on the screen and manually entering data with the onscreen keyboard. None of the panels have any required fields.

Swipe between panels from right-to-left to advance to the next panel.

To move directly to a specific panel tap an item's icon along the navigation tray, and select the desired panel.

Tap the Menu button in the top right corner of any panel to Save, Print and Other actions.

Responder Panel

The Responder panel should populate with pre-entered information about the responder. Verify information is correct to continue. Update any fields that are incorrect, as shown in FIGURE 1-4:

  1. Tap any field to bring up the onscreen keyboard or drop down to update.
  2. Swipe to continue.

Mission Panel

 

The Mission Panel is used for the responder check-in and mission assignment process. Select fields, as shown in FIGURE 1-5:

  1. Set the Deployment Date.
  2. Select the Responder Status.
  3. Enter MA Number (Mission Assignment), if known.
  4. Enter TO Number (Task Order), if known.
  5. Select the Federal Deputation Status.
  6. Select the State LE Authority.

    Tap Equipment Issued if the responder is receiving equipment.

    Type information into the Equipment Comments as necessary.

  7. Swipe to continue.

Other Panel

Using the Other Panel, as shown in FIGURE 1-12:

The HC Standard® Administrator has the ability to customize the Other panel. Here the user can update all pertinent information. This is the final screen in the responder record.

  1. Swipe to automatically save the responder record.
 

Note: The input fields on this panel may vary as the HC Standard® Administrator adds or removes information. There may be instances when this panel is blank which means no additional information needs to be collected.

Next Steps Panel

Using the Save FLEO ID? Prompt, as shown in FIGURE 1-13:

The user will be asked to save the FLEO ID after swiping from the Other screen.

The responder record will automatically be saved and the user decides what action to take next:

  1. Continue with this FLEO ID - Saves and keeps current record open.
  2. Enter a new FLEO ID - Saves the current responder and navigates to the FLEO ID panel.
  3. Return to the main screen - Saves the current responder and returns to the Main Screen.
  4. Cancel - Does not save the record at this time and keeps the record open.

Additional Actions (While in a responder record)

Additional Actions (While in a responder record)

Responder Record Menu

 

The Menu button allows users to perform the following additional functions within a responder record, as shown in FIGURE 1-14:

  1. Scan – Save the current responder record and immediately use the camera to scan another barcode.
  2. Save- Captures a snapshot of the responder and continue working with the responder record.
  3. Save + New - Save the current record and start a new one.
  4. Save + Quit – Save and exit the current record, and return to the Responder Tracking main screen
  5. Close FLEO ID – Discard the current record and return to the Resonder Tracking main screen.
  6. Print – Print the current responder record to a connected Bluetooth Printer.
