Using Sort and Filter

The Sort and Filter button will allow a user to set sort and filter rules for the current record view. Sorting allows data to be viewed in a more comprehensible view. For example, a hospital receiving patients may want to group the incoming patients based on Age range. To apply a sort using the sort and filter button, as shown in :

  1. Click or tap the Sort + Filter button.
  2. Click or tap a drop-down menu to select a choice to filter. Some lists will allow multiple items to be selected.
