Coordinating User

To access Patient Placement as Coordinating User:

  1. Select Coordinating user.
  2. SelectProceed with Training/Exercise to use HC Standard® Patient Placement in training mode.

  3. OR Select Proceed with Real World Event to use HC Standard® Patient Placement in live mode.

 

Note: an Organization cannot be selected when logging in as Coordinating User

Home
