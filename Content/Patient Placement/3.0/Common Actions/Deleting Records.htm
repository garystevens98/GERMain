Deleting a Record

Users with permission may delete records from HC Standard® Patient Placement. To delete a record, as shown in FIGURE 1-2:

  1. Select the check box next to the desired record(s).
  2. Select the Bulk Actions button.
  3. Select the Delete Patients option from the dropdown
  4. Select Cancel to go back without making any changes
  5. Select Delete to remove the record(s)
