Using Quick Filters

Quick filters are the drop-downs at the top of a record view page. These easy access items allow a user to quickly look for certain records according to a particular criteria. To use quick filters, as shown in FIGURE 1-1:

  1. Select The filter icon,

Quick Filters

Then:

  1. Select a Measure.
  2. Select a Query Operation.
  3. Select a List Item.
  4. Select the plus sign to add the criteria.
  5. Select Clear to undo your entries.
  6. Select Apply for the filter to take effect.
