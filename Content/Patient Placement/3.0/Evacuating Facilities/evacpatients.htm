Evacuating Facility Users

Access the desired role from the Home screen, as shown in FIGURE 1-1:

  1. Select the Acute Care Facility or Long Term Care Facility organization type.
  2. Select the Organization.
  3. Select the Proceed with Training/Exercise button to use HC Standard® Patient Placement in training mode or select the Proceed with Real World Event button to use HC Standard® Patient Placement in live mode.

Select Role

Continue accessing the desired role by entering the required information, as shown in FIGURE 1-2:

  1. Enter the Contact Name and Contact Phone Number.
  2. Select Evacuating Facility .
  3. SelectProceed.

Enter Contact Information

The Evacuating Facilities - Dashboard will display, as shown in FIGURE 1-3. The dashboard contains navigation options and summary chart data.
