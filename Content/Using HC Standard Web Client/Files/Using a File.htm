Files

Files can be shared among all HC Standard® users who have been granted this permission by an HC Standard® Admin.

The file function allows access to important files, regardless of the specific computer being used. As long as the file has been uploaded and placed in the user’s workspace and the computer being used has the software to open the specific file type, the file is accessible. Any file type can be uploaded to HC Standard®. At this time, only HC Standard® Admins may add a file.

Viewing a File

To view a file within a Workspace, as shown in FIGURE 1-1:

  1. Select the File tab in the Workspace Item menu.
  2. Select the File name.
  3. The File will open or will provide the option to download the file.

Adding a File to a Dashboard

To add a file to a dashboard, as shown in FIGURE 1-2:

  1. Select the File to be added to a dashboard.
  2. Select the Tools button.

Tools Button

To continue adding the item(s) to a dashboard, as shown in FIGURE 1-3:

  1. Select the circle of the Dashboard that will receive the file.
  2. Select the Continue button to confirm.
