Report Controls

The controls found at the top of the Report preview are explained in FIGURE 1-1.

  1. Page Navigation: Navigate forward and backward through a report.
  2. Print Preview: View how a report will look before printing.
  3. Download: Save the report to Acrobat (.pdf), comma delimited (.csv), Excel (.xls), Rich Text Format (.rtf), Web Archive, XPS Document and Tagged Image File Format file.
  4. Print: Print the report using a printer on the user’s local computer or network.
  5. Zoom: Zoom in or out of the report for readability.
  6. Parameter Options: Opens the parameter options used to generate the report.
  7. Run: Re-run the report.
