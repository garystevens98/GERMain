Using and Viewing a Report

To view a report within a Workspace, as shown in FIGURE 1-1:

  1. Click or tap the Reports tab in the Workspace Item menu.
  2. Click or tap the Report name.

Reports

When the report generates, a user may then see parameter options for the report, as shown in FIGURE 1-2.

  1. Select the appropriate parameter options.
  2. Click or tap the Run button.
