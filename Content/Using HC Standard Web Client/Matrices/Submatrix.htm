Creating a Submatrix

Design and Save a Submatrix

Users with the appropriate permission may be able to create a submatrix for any matrix in a workspace. These submatrices may be shared with other users in the workspace, or created privately for the user.

 

Important! Once created, a submatrix can only be edited or removed from Admin mode.

To create a submatrix, as shown in FIGURE 1-1:

  1. From the desired parent matrix, customize the matrix by adding sort, filter or measure display options.
  2. Select the Create Submatrix button.

Create Submatrix

Edit the submatrix options, as shown in FIGURE 1-2:

  1. Enter a name.
  2. Enter a description, if desired.
  3. Check Is Private to make the submatrix only available to the user creating it.
  4. Check Display All Measures to have all measures displayed by default.
  5. Select the Default View when a user views an entity's details.
  6. Select the Measures tab to make any changes to measure options.
  7. Select the Sorting tab to make any sorting changes.
  8. Select the Filtering tab to make any changes to filters applied to the submatrix.
  9. Select Save to save the submatrix.
  10. Select Back tab to abandon the Add SubMatrix tool.

Create Submatrix Options

Once saved, a submatrix will be available from the matrices list under the workspace. All changes to submatrices must be performed from Admin mode.
