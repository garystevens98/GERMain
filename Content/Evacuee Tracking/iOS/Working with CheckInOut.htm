Working with Check In/Out

Entering Check In/Out Information

Check-In/Out Button

 

Scanning a barcode will start an individual patient record or retrieve existing patient information that has already been entered into the system for that patient.

  1. Tap the Check-In/Out button on the Main Screen, as shown in FIGURE 1-1.

Patient ID

The next step requires the user to enter a patient identification number, as shown in FIGURE 1-2:

  1. Enter a Patient ID. The Patient ID is a unique string assigned to a patient. All information on the patient is stored under this Patient ID and is available for recall and updating at later dates.

There are three ways to enter the Patient ID:

Camera Scan Window

  1. Scan a tag – Attach a tag or bracelet to the patient. From the Patient ID screen, tap the Scan button. Proper technique is to align the window with the vertical center of the barcode, and hold the camera steady, as shown in FIGURE 1-3. If the barcode is not read right away, slightly vary the distance between the camera and barcode. If the device has a camera flash, turn it on in low-light conditions by tapping Light. The Front control switches between the rear- and front-facing cameras. The device will automatically focus on the barcode and will start the patient record when it registers the Patient ID.
 
  1. Driver's License - Scan a driver's license 3D barcode to generate a unique Patient ID based on the DL number and automatically fill-in the Patient's demographic information as it appears on the license. For some states, an older issued license may not properly scan.
 
  1. Manual input – If a tag is not available, it is possible to input a Patient ID manually using the onscreen keyboard. Make sure the Patient ID is unique so that the data from one patient is not mixed with another.
 

Note: The Patient ID is the only required field when starting a patient record. Make sure that each new patient is given a unique Patient ID.

Check In/Out Panels

Once a Patient ID is entered, begin entering patient information by tapping on the screen and manually entering data with the onscreen keyboard. None of the panels have any required fields.

Swipe between panels from right-to-left to advance to the next panel.

To move directly to a specific panel, tap an a panel's icon along the navigation tray.

Tap the Menu button in the top right corner of any panel to Save, Print, or perform Other actions.

Demographic Panel

 

View Demographic Information, as shown in FIGURE 1-4:

  1. This is a read-only panel and displays information gathered during registration. Swipe to continue.

Track Panel

 

Enter Tracking Information, such as Check In Time and Check Out Time, as shown in FIGURE 1-5:

  1. Enter Information as necessary.

     

  2. Tap the Clear Panel button to wipe the panel clean for a subsequent update to Check In/Out information.

  3. Swipe to continue.

Transport Panel

 

Enter Transport information, as shown in FIGURE 1-6:

Transportation destinations, jurisdictions, and unit numbers are pre-designated fields set up by the HC Standard® Administrator.

  1. Enter information as necessary.
  2. Current Departure and Arrival Times can be entered by tapping the Now button. A specific date and time can be selected by tapping the box.
  3. Tap the x to the right of the Departure/Arrival Time boxes to clear and enter a different time.
  4. Swipe to continue.

Next Steps Panel

Using the Save Patient Prompt, as shown in FIGURE 1-13:

The user will be asked to save the record after swiping from the Transport screen.

The record will automatically be saved and the user decides what action to take next:

  1. Continue with this Patient - Saves and keeps current record open.
  2. Enter a new Patient - Saves the current record and navigates to the Type or Scan Patient ID panel.
  3. Return to the main screen - Saves the current record and returns to the Main Screen.
  4. Cancel - Does not save the record at this time and keeps the record open.
 

Note: It is not necessary to complete every panel for a record in order to save a record's information.

Additional Actions (While in a record)

Additional Actions (While in a record)

Record Menu

 

The Menu button allows users to perform the following additional functions within a patient record, as shown in FIGURE 1-14:

  1. Scan Patient – Save the current record and immediately use the camera to scan another barcode.
  2. Save- Captures a snapshot of the record and continue working with the record.
  3. Save + New - Save the current record and start a new one.
  4. Save + Quit – Save and exit the current record, and return to the HC Evacuee Tracking main screen.
  5. Close Patient – Discard the current record and return to the HC Evacuee Tracking main screen.
  6. Print – Print the current record to a connected network Printer.
  7. Audio – Displays any audio files associated with the record.
  8. Photos – Displays any photos associated with the record.
  9. Video – Displays any videos associated with the record.

Capturing Audio Notes

On any panel in a record, audio can be added to supplement the text information. If the user is on a specific section of the application and does not have time to type into the device, the user may capture audio notes that will be associated with that specific page or panel in ETM.

Audio Panel

To create a recording, as shown in FIGURE 1-17:

  1. Tap on the RECORD button and begin speaking into the unit. Tap on the RECORD button a second time to stop recording.

If the user wishes to play back the notes recorded:

  1. Tap the play button for the recording.

When complete with audio recordings:

  1. Tap < Back icon to go back to the previous screen.

Delete a recording by swiping to the left over a recording title then tapping the red circle.
