User Management

To access all Admin Functions:

  1. Click your name in the top right.
  2. Click Admin Mode.

To access users:

  1. Click the people icon on the left of the screen.
  2. Click Users.

 

To reset passwords, delete users, or unlock accounts:

  1. Click the user.
  2. Click the Actions button.

 

 

Note: To add a bookmark to this page click the ribbon as shown below.

 

 

Actions Modal that will appear.

Create new users by clicking the Create New User button.

 

 

Note: Double-click any user to open their user record.

 

 

Required fields include:

  1. Name.
  2. Username.
  3. Email (must be unique and active).

  4. Click the Groups tab in order to assign user permissions.

To add a user to a group:

  1. Click the Group tab from within the user profile.
  2. Click Add Groups.
  3. Select and add the groups below based on user access level.
