New Incident Creation

To access all Admin Functions

  1. Click your name in the top right.
  2. Click Admin Mode

Access lists by:

  1. Clicking the Org Chart Tree icon on the left of the screen.
  2. Click Lists.

Double-click the picklist Incident.

  1. Select the Items tab.
  2. Click Add Picklist Item
  3. Provide a Name for the new Item (the incident name.
  4. Insert the number 1 for Sort Order
  5. Click Add
  6. Click Save

 

 

Note: Bookmark this page by clicking the ribbon at the top left

 

 

To access Matrices

  1. Click the four square icon on the left.
  2. Click Matrices.

Double-click on the matrix name Incident Victims.

  1. Select the Filtering tab.
  2. Click Values dropdown.
  3. Select the incident(s) you do want to see bodies for. You may need to deselect a previous incident
  4. Click Save.
