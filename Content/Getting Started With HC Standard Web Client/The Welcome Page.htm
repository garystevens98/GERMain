The Home Page

After logging into HC Standard®, the Home page appears.

The Home page displays a Tab Navigation Menu at the top of the screen, as shown in FIGURE 1-1. Depending on the selection made within each menu, the bottom of the screen will update to show a dashboard or a workspace.

Home Page

Tab Navigation Menu

The Tab Navigation Menu, at the top of the screen, includes several navigation options. Clicking or tapping on any of these options will present different valuable tools and information within HC Standard®, as shown in FIGURE 1-2.

  1. Application Launcher: The system defaults to HC Standard® 4.0. From the drop-down menu, a user may switch to another application in the HC Standard® Suite (Incident Commander, CHATS Module, CHRS Module, or EPCR Module).
  2. Home: Access the Welcome Page from anywhere in the application.
  3. Notifications: View received notifications and send new notifications to other users. For more information, See "Notifications " on page 1.
  4. Dashboards: View and navigate available dashboards. For more information, See "Dashboards" on page 1.
  5. Workspaces: View and navigate available workspaces. For more information, See "Workspaces Overview" on page 1.
  6. Bookmarks: Access bookmarked items. For more information, See "Bookmarks" on page 1.
  7. Help and Support: Access HC Standard® online help and support.
  8. User Profile Settings: Change options for the user profile, general settings, enter Admin mode (if available), enter full screen, and logout.

Desktop vs Mobile View

HC Standard® 4.0 introduces a new responsive design to accommodate both desktop and mobile screens. Depending on the type of device used, users may see small differences between each respective view. Overall, there are no limitations to using either view, only visual aesthetic differences are made.

FIGURE 1-3 shows HC Standard® as seen on a desktop screen. HC Standard® utilizes all available real estate to display the application. The workspace item navigation menu is displayed along the left-hand side of the screen.
