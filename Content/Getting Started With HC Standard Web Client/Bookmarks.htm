Bookmarks

To make navigation easier in HC Standard®, important items can be bookmarked for quick access. In FIGURE 1-1 below, a map is being bookmarked inside a workspace for quick access. To bookmark an item:

  1. Navigate to the location within HC Standard® that is to be bookmarked. In this example, the user is navigating to Maps.
  2. Select the Bookmark icon in the navigation bar. In this example, the bookmark icon is located to the left of the PTS link.

Bookmarking a Location

Finish setting up the bookmark, as shown in FIGURE 1-2.

  1. Type a Name for the bookmark in the text box.
  2. Select the Add button to save the bookmark.

Add Bookmark Name

To view all bookmarks, as shown in FIGURE 1-3:

  1. Select the Bookmarks tab in the Tab Navigation Menu.
  2. Begin typing in the search bookmarks field to narrow the results of all bookmarks. In the example below, there is only one bookmark.
  3. Select the bookmark name to navigate directly to that bookmark's location.
