Getting Started with HC Standard® Administrator Mode

Entering Admin mode

User accounts designated with HC Standard® Sub-Administrator permissions will have access to HC Standard® Admin mode. After the initial login is completed, Admin mode can easily be switched on and off within the HC Standard® application. In the example below, the user is entering Admin mode. To access HC Standard® Admin mode, as shown in FIGURE 1-1:

  1. Click or tap the User icon.
  2. Click or tap the Admin mode option located in the drop-down menu.

Admin mode in User Tab

 

Important! If Admin mode is unavailable, the appropriate administrator permissions have not been assigned.
