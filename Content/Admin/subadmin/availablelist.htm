Selecting Items from an Available List

In different locations in Admin mode, it is possible to select items from a list of all objects of a certain type that are currently available in the system. FIGURE 1-1 shows an example of how a Sub-Admin can assign a user to multiple user groups. Instructions for accessing tools, like the Edit User tool, are provided in their respective sections within this manual.

 

Note: Items are added to or removed from the list the same way in all scenarios.

To add available items to the new list, as shown in FIGURE 1-1:

  1. Select on the Add <Item> button to open the Add <item> dialog box. In FIGURE 1-2, the Add <Item> button is labeled Add Matrices.

Add Matrices Button

In FIGURE 1-2, user groups are being added or assigned to a user.

  1. Select the circle for each item to be added.
  2. Select the Add Selected button.
  3. Select the Done button to close the Add <Item> dialog box.

Add Items

In FIGURE 1-3, a user group is being removed from the user. To remove items from a list:

  1. Select the circle for each item to be removed.
  2. Select the Remove Selected button.

Remove Selected Items
