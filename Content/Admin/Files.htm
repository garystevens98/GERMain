Files

The Files tool allows for the uploading of almost any type of file into the system. These files then become available to be placed in any number of workspaces.

To manage files, as shown in FIGURE 1-1:

  1. Select on the Workspaces & Workspaces Items icon.
  2. Select on the Files icon under the Manage My Workspaces section.

Files

FIGURE 1-2 shows the main Files page. This page lists all existing files, along with the description provided for each, if any.

Uploading a New File

Files can be added to the database from any device connected to a user's computer, such as a local hard drive, a flash drive, or network location.

To add files, as shown in FIGURE 1-3:

  1. Select the Upload Files button.

Upload Files Button

Selecting this button will display the Upload Files tool, showing the files on the local hard drive that can be selected for upload, as shown in FIGURE 1-4.

To upload a file:

  1. Select on the Choose Files or Browse button. Navigate to the location where the newly uploaded file is stored. Select the newly uploaded file and Select the Open button.

To return to the Upload tab without selecting a file, Select the Cancel button. Clicking or tapping the Open button will cause the file name to be shown in the Upload tab.

  3. Enter a description for the file, if desired, in the Description field.
  4. Select the Workspaces tab. Select Workspaces from the available groups to place the file. For more information, see "Selecting Items from an Available List" on page 1.
  5. Select the Upload All button to upload the file.

  6. Select the Back button.
