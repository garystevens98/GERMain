Dashboards

Dashboards allow an administrator to set up a shared dashboard for a group of users. These dashboard assignments are managed through Dashboards.

To manage group dashboards, as shown in FIGURE 1-1:

  1. Select on the Users and Permissions icon.
  2. Select on the Dashboards icon under the Manage My Users section.
