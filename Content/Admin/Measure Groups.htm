Measure Groups

The Measure Groups tool provides the ability to gather any number of measures together under a common label. For example, a patient's measure group would include Name, Age, Gender, and Triage Status measures.

To manage measure groups, as shown in FIGURE 1-1:

  1. Click or tap on the Entities icon.
  2. Click or tap on the Measure Groups icon under the Manage My Entities section.

Measure Groups

FIGURE 1-2 shows the main Measure Group page.  This page lists all existing measure groups, along with their names and descriptions.

Creating a New Measure Group

To create a new group, as shown in FIGURE 1-3:

  1. Click or tap on the Create New Measure Group button.

Create New Measure Group Button

The Add Measure Group tool displays with the Details tab already selected, as shown in FIGURE 1-4.

  1. Enter a name for the group and a description, if desired.

Creating Measure Group - Details

To associate individual measures with the new group, as shown in FIGURE 1-5:

  1. Click or tap on the Measures tab.
  2. Click or tap the Add Measures button. Select the desired measures from the available list to add them to the measures in the group. For more information, see Selecting Items from an Available List on page 1.
