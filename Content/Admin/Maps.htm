Maps

Maps associate data with geographical locations. The ability to view data visually on a map is an invaluable tool. Critical statuses such as Hospital Status, Blood Availability, and Pharmaceuticals can all be mapped out.

To manage maps, as shown in FIGURE 1-1:

  1. Select on the Workspaces & Workspaces Items icon.
  2. Select on the Maps icon under the Manage My Workspaces section.

FIGURE 1-2 shows the main Maps page. This page lists all existing maps.

Creating a New Map

To add a new map, as shown in FIGURE 1-3:

  1. Select the Create New Map button.

Create a New Map Button

The Add Map tool will display with the Details tab already selected, as shown in FIGURE 1-4.

  1. Enter a Name and a brief Description, if desired.

Add Map - Details

To assign the map to a workspace, as shown in FIGURE 1-5:

  1. Select on the Workspaces tab to add a workspace to the map.
  2. Select Add Workspaces button. Assign one or more workspaces to the map. For more information, see "Selecting Items from an Available List" on page 1.

Add Map - Workspaces

To include overlays in addition to matrix data, such as weather radar or time zone, as shown in FIGURE 1-6:

  1. Select the Standard Layers tab.
  2. Select the Add Standard Layers button. Select any Standard Layers that should be included in the map. For more information, see "Selecting Items from an Available List" on page 1.

Adding a Map - Standard Layers

To determine matrix data that should be included on the map, as shown in FIGURE 1-7:

  1. Select the Matrix Layers tab.
  2. Select the Select Parent Matrix button. Add a parent matrix or multiple matrices that contain the data that is to be mapped. For more information, see "Selecting Items from an Available List" on page 1.
  3. Use the Icon column to set customized icons for each map layer.
  4. Use the Measures column to set what kind of measure information will be displayed with the location data (this information will be displayed when a location is clicked/tapped).

Add a Map - Matrix Layers
