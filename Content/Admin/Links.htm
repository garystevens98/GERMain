Links

The Links tool allows website URLs to be saved as objects that can be placed in workspaces. The website links can be launched and viewed within the HC Standard速 operating environment.

To manage links, as shown in FIGURE 1-1:

  1. Select on the Workspaces & Workspaces Items icon.
  2. Select on the Links icon under the Manage My Workspaces section.

Creating a Link

To add a new link, as shown in FIGURE 1-3:

  1. Select on the Create New Link button.

Create New Link Button

The Add Link tool will display with the Details tab already selected, as shown in FIGURE 1-4.

  1. Enter a Name for the link, and select a Type.
  2. Enter the website address into the URL field.
  3. Select the URL type.

Types of URLs:

  • URL in tab: Opens the link in a new browser tab. A user's browser will need to have pop-ups enabled for the HC Standard速 site to view this link type.
  • URL in frame: Opens the link inside a frame in HC Standard速 . Unsecured websites will not display unless mixed content settings are enabled in a user's browser settings.
  • URL with session token in frame: Enables single sign-on for HC Standard速 applications.For GER use only.
  • Twitter feed: If entering a Twitter feed, use this type to properly display the feed.
  • Camera: Sentinal AVE camera links can be used with this type to overlay camera controls on the video. Other camera feeds may be added using URL in tab or URL in frame type.

  1. To associate individual workspaces with the new link, Select on the Workspaces tab. Select an available workspace from the list. For more information, see "Selecting Items from an Available List" on page 1.
  2. To ensure the link is correct after entering the URL, Select the Preview tab.
