Logs

The Logs tool gives a complete listing of the log table pulled directly from the sequel server database for the application. Generally, this tool is used by a GER Technician when troubleshooting the system.

To view logs, as shown in FIGURE 1-1:

  1. Select on the System icon.
  2. Select on the Logs icon under the Manage My System section.

Logs

FIGURE 1-2 shows the main Logs page. This page lists all logs collected by the system. Admins do not manage anything on this page.

Download Logs CSV

Admins may download a CSV log of all entries in the system.

To download a CSV, as shown in FIGURE 1-3:

  1. Select the Download CSV button and the CSV file will download.
