Maps

Introduction

The map screen shows data in a geospatial view. Users have the ability to filter data, view heat streams, and annotate markings or points of interest within the map.

Access the Map app from the application menu, as shown in FIGURE 1-1:

  1. Select the Map button.

Application Menu - Map

Map Actions

Maps may be customized to suit the user's needs by using map actions. Map actions are available to the user, as shown in FIGURE 1-2:

  1. Map Layers -allows the user to toggle on or off different map layers, data layers, and custom user layers.
  2. Find Location - .searches the map for the address or keyword entered.
  3. Current Location - zooms the map down to the user's current location. For best results, use with a GPS enabled device.
  4. Full Screen -toggles fullscreen mode.
  5. Reset View - returns the map view to the default state.

Map Actions

Map Controls

The map controls are along the right side of the display, shown in FIGURE 1-3:

  1. Zoom In/Out - zooms the map in or out.
  2. Annotation Tools - provides the user with objects to add to a map.
  3. Annotation Edit Mode - allows the user to make changes to annotations.
  4. Waterfall List - shows a listing of items within view on a map. As a user makes changes to filters or map view, the list will filter to show relevant items. Select an item in the list to navigate the map to the item's location.

Map Controls

Annotate A Map

To make an annotation on the map, as shown in FIGURE 1-3:

  1. Select an annotation button.
  2. Select within the map and drag to draw the annotation as desired.

Annotate Map

The following options appear after drawing an annotation and are entirely optional, as shown in FIGURE 1-5:

  1. Add a caption for the annotation.
  2. Check Use as geofence to filter data in the Waterfall List to only show items from within the annotation's borders. When enabled, this filter item will display in the top left corner of the map.
  3. Select the X to close the dialog.

Annotation Options

  1. Select the edit annotation button.
  2. Select a border of the annotation and make desired change.
  3. Select Save to commit changes.

    OR

Map Layer Types

The following table outlines the different map layer types available in HC Analytics:

Map Layer Types
Type

Description

DS7
  • The system will reach out to a DS7 server based on the config code, username, and password provided in the DS7 login popup.
  • Types of objects read in from DS7 include articles, assets, events, incidents, indicator reports, and team locations
ESRI Feature Service
  • The ESRI feature service layer type will allow the user to add features served by a ESRI ArcGlS server in feature service form.
ESRI Map Service
  • The ESRI map service layer type will allow the user to add features served by a ESRI ArcGlS server in map service form.
  • These are tile-based rendered images that are drawn over the map. The layers attribute is an array of integers like [0,1,2,3]. used to define which layers in the map service are to be used. An optional token may be sent if the server uses token-based security.
GeoRSS
  • GeoRSS feeds need a name and a URL for a GeoRSS feed available on the internet. This type of RSS feed contains additional tags with latitude and longitude information for each item. Optionally, an internet-accessible URL for an icon image may be added as well, which will show on the map in place of each item.
KML
  • KML layers are defined by entering a name and an internet-accessible URL for the KML file. The extension must be KML. KMZ is not supported at this time.
Map Box
  • Map Box is an internal HC Analytics GIS layer.
WMS
  • Web Map Service (WMS) is a popular format for sharing tile--based image layers.

Filter Map Data

HC Analytics analyzes and displays thousands of feeds instantaneously. Users have two methods to filter data in a map:

  1. Using annotations to apply a geofence, see "Annotate A Map" on page 1
  2. Using map filters to filter by tag or date range.

Apply Map Filter

To apply a map filter, as shown in FIGURE 1-13:

  1. Select the Filter button.

Filter Button

Apply filter options, as shown in FIGURE 1-14:

  1. Check Tags and enter a tag.
  2. Add additional tags, if desired.
  3. Select Save to save the filter.

    OR

  4. Select Close to cancel changes.

Filter Options

  1. Select the X on the filter item.
