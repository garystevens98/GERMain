Using HC Mobile - Community Paramedicine

Launch HC Mobile™

  • Tap the HC Mobile icon to launch HC Community Paramedicine.

Login to HC Mobile™

FIGURE 1-98: Login Screen

 

Log in to begin adding patient records, as shown in FIGURE 1-98.

  1. Enter Username and Password as provided by an HC Standard® Administrator.
  2. Tap the Login button.

    OR

  3. If username and password are inaccessible, the user can tap the Skip Login button to enter the HC Mobile™ application. Patients can be entered into HC Mobile™ under the Community Paramedicine application (CPM) at this point; however, no data will synchronize to the server until proper user credentials are entered.

 

FIGURE 1-99: Module Selector

The modules available to the user will display on the user’s device, as shown in FIGURE 1-99.

  1. From the application launcher, select HC Community Paramedicine.

If it is the first time launching the HC Community Paramedicine (CPM) application and there is a data connection after logging in, the Synchronization screen will appear, as shown in FIGURE 1-100. Once the synchronization process is complete it will display the main screen. If a user wishes to return to the main screen sooner:

FIGURE 1-100: Sync Screen
  1. Tap the Tracking icon in the navigation tray to continue.

If there is no data connection or the CPM application is still open from a previous log in, the user will be directed to the HC Community Paramedicine Main Screen.

Community Paramedicine Main Screen

FIGURE 1-101: Main Screen

The Community Paramedicine Main Screen, as shown in FIGURE 1-101

  1. Connection Status - Shows current connection status to the server.
  2. Sync Status - Shows last date and time the device synchronized. Records on device shows how many records are currently on the user's device. A green icon means there are no records and a yellow icon indicates there are still records to be synchronized.
  3. Current Information - Shows current Area of Care and Division settings.
  4. Record Button - Starts desired entry mode.
  5. GPS - Shows current GPS data.
  6. Sync - Controls data synchronization.
  7. About - Displays HC Community Paramedicine version information.
  8. Summary - Presents memory, battery and GPS status.

If Area of Care and Division have not been set, then upon Record entry, the user will be prompted to enter the appropriate Area of Care and Division information. Entering this information will only be necessary the first time using the device for a specific session. Once this information has been entered, the user will be able to enter data and will not need to go through this step again. For more information, See "Area of Care/Division"
 

Note: The device may occasionally lose its connection with the server. If this occurs, patient records will remain on the device’s internal memory but will not be uploaded to the database. The device should Auto Synchronize with the database once the connection is re-established.

HC Community Paramedicine Menu

FIGURE 1-102: Menu

 

To access the menu, as shown in FIGURE 1-102:

  1. Tap the Menu button to access the menu.

FIGURE 1-103: Menu Items

Menu Items include, as shown in FIGURE 1-103:

  1. Change AoC/Division – Update Area of Care, and Division information. Tap this to go to the AoC/Division panel.
  2. Account Settings - Access settings for the user account.
  3. Logout - Log out of the HC Standard® account.
  4. Switch Module - Switch to another HC Mobile™ application.

Account Settings

FIGURE 1-104: Account Settings

 

Users are able to access account settings from the Menu, as shown in FIGURE 1-104.

  1. Enter or change the EMAIL address for the account.
  2. Enter or change the PHONE NUMBER associated with the account.
  3. Tap Change Password to change the password for the account.
  4. Tap the <Back button to go back to the previous page.

FIGURE 1-105: Account Settings

To change the password, as shown in FIGURE 1-105:

  1. Enter the OLD PASSWORD.
  2. Enter the NEW PASSWORD.
  3. Enter the new password again under CONFIRM PASSWORD.
  4. Tap Change Password to save changes.
  5. Tap Cancel to abandon changes.

Area of Care/Division

The Area of Care/Division panel, as shown in FIGURE 1-106, is where the user enters key elements describing the operation of HC Community Paramedicine. This only needs to be done once per session.

Some items can be selected from drop-down lists which the HC Standard® System Administrator has created in HC Standard®.

FIGURE 1-106: Area of Care/Division Panel
  1. AREA OF CARE - Tags each record scanned with this information.
  2. DIVISION - Tags each record scanned with this information.
  3. Tap Done once the user has made the appropriate selection(s).
