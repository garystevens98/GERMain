What Is New In HC Standard®4.4?

HC Standard® 4.4 features several new and improved features. Some highlights in HC 4.4 include the following.

Persistent User Settings

When users make changes to the layout of matrices, maps and dashboards, those items will no longer revert to their defaults at the end of a session. All of the choices a user made will remain when the user next logs into the system. A user can choose to return all of the settings to their defaults using the Reset Settings button on the Edit Profile panel, or to return the display settings on a particular matrix to their defaults using the Reset button on the matrix Display dropdown.

Matrix Sections

Measures associated with an entity group can now be placed into any number of different sections. Matrices based on the entity group will automatically display the measures in their assigned section in the entity measures view. Users can also choose to view all measures at once. Unassigned measures are displayed under the heading Overview.

Manipulate Measures in Matrices

Users can now not only choose to display all available measures by selecting Display All with a single click or tap, they can choose to Display None with a single click or tap, too.

Additionally, users can now re-order measures in a matrix by clicking or tapping a measure and dragging it to the desired location in a matrix.

Add GPS Coordinates from Web

Users can add now add GPS coordinates to a record using the web application. For more information, See "Using a Matrix"

Submatrices

Users with the appropriate permissions can now create either a private or public submatrix directly from an existing matrix, rather than having to build each new matrix from scratch. Users can view data as it will actually appear in the new submatrix by setting sort, filter and display preferences in the existing matrix. When the settings are correct, clicking Create SubMatrix will launch the Add SubMatrix tool with those settings pre-populated on the appropriate tabs.

Save and Next While Editting Entity Data

Instead of being required to save data changes made to an entity and then move to the next entity, users can now save and move to the next entity in the matrix with a single click of the Save and Next button

Access Rules - Notifications

Users and administrators with the appropriate permissions can now send a single input-required notification using a matrix to which individual users have only partial access. If access rules are applied, each recipient of the notification will be able to enter data only for the entities they have permission to see and edit..

Pin Clusters in Maps

When there are too many pins in a certain area of a map for all of them to be displayed at the current magnification, HC Standard® will display hexagonal pin clusters that tell the user how many pins are in that area: Small green icons for 2 to 10 pins, medium-sized yellow ones for 11 to 100 and large red ones for more than 100.
