System Requirements

HC Shelter Tracking users, granted appropriate permissions, may access HC Shelter Tracking over the Internet using any device that has the following features:

System Requirements

PC

  • Microsoft Internet Explorer 10 (or higher); or the latest version of Google Chrome or Mozilla Firefox
  • Adobe Acrobat Reader 6.0+ for Forms; other PDF reader software may be used but is not supported
  • 1.6 gigahertz (GHz) or faster 32-bit (x86) or 64-bit (x64) processor
  • Minimum 1gb (or more) RAM
  • Internet Access

Mobile Device

  • iOS 5+
  • Android 4+
