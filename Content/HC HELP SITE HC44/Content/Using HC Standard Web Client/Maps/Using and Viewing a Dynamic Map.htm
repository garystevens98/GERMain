Viewing a Map from the Matrix

In a Workspace

Click the Matrix with the data that User wants to see on a Map

Filter the data in the Matrix as necessary.

Click the Map icon (bottom, far right icon.)

FIGURE 1-72: Matrix Map View

 

Users may change what data displays on the Map by using the drop-down in the upper right corner of the Map.
