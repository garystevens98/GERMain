Matrices

Matrices are the heart of the system because data is entered and updated here. The visualization of data also takes place here through the creation of reports, maps and charts.

A Matrix is a view of the data in a grid format containing various measures for a specific set of entities or things.

 

Important! A working knowledge and understanding of this tool is required to be able to use the system.
