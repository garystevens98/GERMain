Dashboards

As more information is added to a workspace in HC Standard速, the ability to place the most important and useful information on a customized Dashboard becomes extremely helpful. This allows for a unique visual representation of the data in an easy to use and view interface, as shown in FIGURE 4-97. Dashboard views will refresh automatically, displaying the workspace items previously selected.

FIGURE 4-97: Dashboard

 

Note: The HC Standard速 Admin must provide permission for dashboard access. If the appropriate dashboard privileges are not supplied, the user's Dashboard option will not appear.

The HC Standard速 Dashboard is customizable at the individual user level. The information, data, and format can be selected for each Dashboard.

To view a dashboard, as shown in FIGURE 4-98:

  1. Click or tap the Dashboard tab at the top of the screen.
  2. Click or tap the desired dashboard Name to open.

FIGURE 4-98: Dashboards Available To Open

 

Note: Any workspace item may be selected and displayed on the dashboard.

Using Dashboard Favorites

To aid in navigation of HC Standard速, users may favorite commonly accessed dashboards.

To tag a dashboard as a favorite, as shown in FIGURE 4-99:

  1. Click or tap the Dashboard tab in the Tab Navigation Menu at the top of the screen.
  2. Click or tap the star icon next to the desired dashboard name. Click or tap the star icon again to remove it as a favorite.

FIGURE 4-99: Dashboard Favorite Icon

To filter the dashboard view to only show favorites, as shown in FIGURE 4-100:

  1. Click or tap the Dashboard tab in the Tab Navigation Menu.
  2. Click or tap the Show Favorites toggle in the dashboard tab.

FIGURE 4-100: Dashboard Favorites Toggle
