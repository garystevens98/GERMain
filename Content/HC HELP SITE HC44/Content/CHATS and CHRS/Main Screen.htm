Using HC Standard® Hospital Alert Tracking

HC Hospital Alert Tracking Main Screen

Depending on the permissions assigned to the user, the HC Hospital Alert Tracking homepage will display the following, as shown in FIGURE 2-1:

  1. Ongoing Alerts - displays the number of current alerts.
  2. Current | Recent - toggle that shows the current status of facilities. Switching to Recent shows a chronological list of status changes.
  3. Filter - changes the current facility view (if available).
  4. Tabs - cycles between different types of facilities (if available) or view reports.
  5. Facility List - lists facilities and current time on a particular status. Hovering over a status will display more information.
  6. Edit icon - allows the user to request or change the status of a facility (with the appropriate permission).

FIGURE 2-1: Main Screen

Changing a Facility Status

Users with the appropriate permissions can change a facility status, as shown in FIGURE 2-2:

  1. Click or tap the Edit icon for the desired facility.

FIGURE 2-2: Edit Status

Edit the request, as shown in FIGURE 2-3:

  1. Select the Request.
  2. Select the Reason.
  3. Click or tap Submit to send the request.

OR

  1. Click or tap Close to cancel the request.

FIGURE 2-3: New Request

After a request has been made, a user may cancel the request, as shown in FIGURE 2-4:

  1. Click or tap Cancel.
