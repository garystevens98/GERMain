Events

Introduction

Events are descriptions of activities that happen at a specific time, such as a party or parade. Events may be located, associated with assets, and given points of contact. For special event operations, events are an important part of tracking overall activity.

Access the Events app from the application menu, as shown in FIGURE 1-43:

  1. Click or tap the Events button.

FIGURE 1-43: Application Menu - Events

Events List Screen

The events list screen will display, as shown in FIGURE 1-44:

  1. Event Type - filters the list to desired type.
  2. Reset Filters - clears filters applied.
  3. Search - filters the list view to the entered value.
  4. Select Action - takes action on items, such as delete.
  5. Display - changes the number of items to display per page.
  6. Page Selection - navigates to a particular page.

FIGURE 1-44: Events List

The event list toolbar contains the following buttons, as shown in FIGURE 1-45:

  1. Create Event - creates a new event.
  2. Event Settings - displays an option for periodic refresh.
  3. Mini Dashboard - shows a mini pie chart of events based on Type or Priority.

FIGURE 1-45: Event List Toolbar

Create An Event

To create a new event, as shown in FIGURE 1-46:

  1. Click or tap the Create Event [+] icon in the toolbar.

FIGURE 1-46: Create Event Button

Complete the event creation, as shown in FIGURE 1-47:

  1. Enter a Name. This is a required field.
  2. Enter a Start Date. This is a required field.
  3. Enter an End Date. This is a required field.
  4. Enter a Type, if desired. Type is used to further categorize an event. For example, a festival parade could be typed as parade.
  5. Enter an Importance, if desired.
  6. Associate Assets, as needed.
  7. Enter a Venue.
  8. Enter Tag and click or tap Add to add tags. Tags are keywords that can be searched and filtered on a map.
  9. Click or tap Save to save the Event.

FIGURE 1-47: Event Overview
