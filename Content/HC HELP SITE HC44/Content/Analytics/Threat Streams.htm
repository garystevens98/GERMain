Threat Streams

Introduction

Threat Streams ingest real-time unstructured threat-related data, process and score the text, and present it to the user in a prioritized manner. This allows users to monitor social media and news for emerging threats and items of concern. Hundreds of thousands of text items are ingested every day.   1. Click or tap the Map button.

Threat Stream Filters

Filters are used on each column to determine which set of articles are shown. Filters are chosen through the drop-down on the left side of the column header.

Add Threat Stream Filters

Threat Stream filters allow a user to find threats specifically by Type, Tags, Phrases, and Location, as shown in FIGURE 1-274:

  1. Click or tap the Topic Filters button.
  2. Click or tap the Add New... item.

FIGURE 1-274: Add New Filter

The Threat Stream Filter dialog will appear, as shown in FIGURE 1-275:

  1. Enter a Name.
  2. Enter Phrases to filter on. Filtering with phrases is more than just finding items that contain the exact string in the phrase. HC Analytics searches items for matches using Natural Language Processing. The search is not case-sensitive and will match on the complete stemmed word. So if the phrase is blueberry, then it will not match items containing blue, but it will return items that contain blueberry or blueberries. If any phrase in the filter is found in an item then it will be returned in the threat stream. HC Analytics filters the whole phrase only. To eliminate results, use "not" in the search query.
  3. Click or tap the Type to select which type of stream to apply the filter.
  4. Add Feed tags. This limits the results to only items that come from a feed that has at least one of the same tags.
  5. Click or tap Location to add a geographic radius to filter data.
  6. Add Assets as desired. When viewing an asset profile page, this threat filter will appear there.
  7. Click or tap Save to save.

    OR

  8. Click or tap Cancel to abandon the dialog.

FIGURE 1-275: Threat Stream Filter

Manage Articles

Articles are displayed in columns depending on the filter and sort applied. Articles are coloured based on the system determined threat level.

Articles tools are displayed in FIGURE 1-278:

  1. Open - opens the article source in a new tab.
  2. Edit - opens the annotation editor.
  3. Details - toggles additional information about the article.
  4. Mark Important - saves the article for viewing under the Starred filter.
  5. Hide - removes the article from the list.
  6. Share - opens a dialog to share the article via email.
  7. Create Incident - opens the Create Incident dialog to start the process of creating an incident based on the article.

FIGURE 1-278: Threat Stream Article

Annotate Articles

If HC Analytics missed a term that may be important as an entity, the user may annotate the article using the Annotation Editor. The annotation editor will also show the user other words that HC Analytics has marked automatically. Users may edit these terms as well.

To open the Annotation Editor, as shown in FIGURE 1-281:

  1. Click or tap the Edit link.

FIGURE 1-281: Edit Article Annotation

The Annotation Editor will display. To add a new entity, as shown in FIGURE 1-282:

  1. Highlight the term.
  2. Select the category for the entity.
  3. Click or tap remove annotation to cancel adding a new entity.

FIGURE 1-282: Annotation Editor - New

To edit an existing entity, as shown in FIGURE 1-283:

  1. Click or tap the term.
  2. Select the a new category, if desired.
  3. Click or tap remove annotation to delete the entity.
