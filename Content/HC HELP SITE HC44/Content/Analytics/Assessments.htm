Assessments

Introduction

Assessments are useful to make sure certain safety and security standards are met on assets. Many jurisdictions use assessments to keep track of vulnerabilities and risks to understand what needs to be improved to increase security.

The Assessments App allows users to perform assessments against assets in the catalog. The assets are driven by imported templates. The assessments are a list of sections, each of which contain questions about the assets. Templates may be tenant-specific.

Access the Assessments app from the application menu, as shown in FIGURE 1-238:

  1. Click or tap the Assessments button.

FIGURE 1-238: Application Menu - Assessments

Assessment List Screen

The Assessment list screen will display, as shown in FIGURE 1-239:

  1. Status - filters the list to desired status.
  2. Type - filters the list to the desired type.
  3. Reset Filters - clears filters applied.
  4. Filter assessments - filters the list view to the entered value.
  5. Select Action - takes action on items, such as delete.
  6. Display - changes the number of items to display per page.
  7. Page Selection - navigates to a particular page.

FIGURE 1-239: Assessment List

The Asset List toolbar contains the following buttons, as shown in FIGURE 1-240:

  1. Create Assessment - creates a new assessment.
  2. Settings - displays options to show new aseessments in the list.
  3. Mini Dashboard - shows a mini pie chart of assessments based on Status or Type.
  4. Print - shows a print friendly page with filter controls.
  5. Aggregate Report - creates an aggregate by question report based on assessments completed with a particular template.
  6. Import/Export - contains options to export an assessment for offline editing or importing a completed offline assessment.

FIGURE 1-240: Assessment List Toolbar

Create An Assessment

To create a new assessment for an asset, as shown in FIGURE 1-241:

  1. Click or tap the Create Assessment [+] icon in the toolbar.

FIGURE 1-241: Create Assessment Button

Edit assessment options, as shown in FIGURE 1-242:

  1. Click or tap the Template Type list and select an item.
  2. Enter Start and End dates.
  3. Click or tap the Assign myself button. Do not enter a custom email address here.
  4. Add one or more previously created Assets. More than one asset may be added and a unique assessment will be created for each.
  5. Enter Tag and click or tap Add to add tags. Tags are keywords that can be searched and filtered on a map.
  6. Click or tap Save to continue.

FIGURE 1-242: Create Assessment

View An Assessment

Once an assessment has been created, the user may view and complete the assessment at their discretion.

To view an assessment, as shown in FIGURE 1-243:

  1. Click or tap the assessment name.

FIGURE 1-243: Asset List

The individual assessment view presents a list of sections on the left, and the main question content in the main area. These questions are multiple choice, single choice, text input, and number inputs. Each question may have an attached image or optional comment text.

FIGURE 1-244: Assessment View

To navigate and use an assessment, as shown in FIGURE 1-245:

  1. Click or tap a Section to navigate to particular set of questions.
  2. Enter data for each question.
  3. Click or tap the Comment icon to add comments to a response. Comments will appear in reports.
  4. Click or tap the Camera icon to attach or remove photos or documents to the response. These files will appear in reports.
  5. After completing each section, or question, click or tap Save to save the data entered. Navigating between different sections will automatically save.
  6. When a section is complete, click or tap the slider to mark the section complete.
  7. Once the every section is marked complete, click or tap the Submit button. Once submitted, the assessment will display a Risk Score in the overview page.

FIGURE 1-245: Edit Widget

Import or Export An Assessment Template

Assessment templates may be exported and completed offline. Templates save as an Excel file type. Once completed, the excel files may be imported into HC Analytics.

Open the import and export options, as shown in FIGURE 1-246:

  1. Click or tap the Import/Export button.

FIGURE 1-246: Import/Export Button

Export An Assessment Template

To export an assessment template, as shown in FIGURE 1-247:

  1. Click or tap the Template to export.
  2. Click or tap the Download template button.

    Complete the Excel sheet.

FIGURE 1-247: Export Options

To import a completed assessment template excel file, as shown in FIGURE 1-248:

  1. Click or tap the Attach file button. Navigate and select the file to import.
  2. Click or tap the Upload template button to import the file into HC Analytics.
