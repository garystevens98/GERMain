Incidents

Introduction

The Incidents app allows users to log incidents. Incidents are threat events which may impact operations or assets. Incidents capture the important information about threat events such as location, date, and pictures of the event. Managing incidents helps an organization keep track of the threat picture around their area of concern.

Access the Incidents app from the application menu, as shown in FIGURE 1-428:

  1. Click or tap the Incidents button.

FIGURE 1-428: Application Menu - Incidents

Incidents List Screen

The incidents list screen will display, as shown in FIGURE 1-429:

  1. Incident Type - filters the list to desired type.
  2. Status - filters the list to the desired status.
  3. Importance - filters the list to the desired importance.
  4. Reset Filters - clears filters applied.
  5. Search - filters the list view to the entered value.
  6. Select Action - takes action on items, such as delete.
  7. Display - changes the number of items to display per page.
  8. Page Selection - navigates to a particular page.

FIGURE 1-429: Incidents List

The incidents list toolbar contains the following buttons, as shown in FIGURE 1-430:

  1. Create Incident - creates a new incident.
  2. Event Settings - displays options for periodic refresh, alerting on new incidents, and showing threat stream items on an incident.
  3. Mini Dashboard - shows a mini pie chart of events based on Type or Priority.
  4. Print - shows a print friendly page with filter controls.

FIGURE 1-430: Incident List Toolbar

Create An Incident

There are two ways to create an incident:

  1. From an article in the threat stream, See "Threat Streams"
  2. From within the Incidents app.

To create a new incident from the Incidents app, as shown in FIGURE 1-431:

  1. Click or tap the Create Incident [+] icon in the toolbar.

FIGURE 1-431: Create Incident Button

Complete the incident creation, as shown in FIGURE 1-432:

  1. Enter a Name. This is a required field.
  2. Enter a Start Date. This is a required field.
  3. Enter an End Date.
  4. Enter a Status, if desired.
  5. Enter an Importance , if desired.
  6. Click or tap the Assign myself button.
  7. Associate Assets, as needed.
  8. Associate other Incidents, if needed.
  9. Enter Tag and click or tap Add to add tags. Tags are keywords that can be searched and filtered on a map or threat stream.
  10. Click or tap Save to save the incident.

FIGURE 1-432: Incident Overview
