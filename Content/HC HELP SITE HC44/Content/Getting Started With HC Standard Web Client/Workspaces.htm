Workspaces

Users will spend most of their time in a Workspace that contains the relevant Workspace Items they want to access. Depending on how user privileges have been set in the system by the HC Standard® System Administrator, users may see a few items, e.g. Links or Matrices, or many items and be able to display information and/or make changes. For example, a user may be able to view Reports, but may not be able to enter data or create Reports.

 

 

Contact the HC Standard® System Administrator for access to additional Workspace Items or privileges.
