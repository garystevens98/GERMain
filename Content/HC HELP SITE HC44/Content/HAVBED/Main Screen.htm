USING HC Standard® HAvBED

HC HAvBED Main Screen

Depending on the permissions assigned to the user, the HC HAvBED homepage will display the following, as shown in FIGURE 1-119:

  1. Request Updates - used to send a notification to users to update HAvBED data.
  2. Column Settings - select which columns to display.
  3. Expand - automatically adjusts column widths to fit the user's browser.
  4. Send to HHS - will send the current HAvBED data to HHS.

FIGURE 1-119: Main Screen

Requesting Updates

Users with the appropriate permissions can request facility updates, as shown in FIGURE 1-120:

  1. Click or tap the Request Updates button.

FIGURE 1-120: Request Updates Button

Edit the request, as shown in FIGURE 1-121:

  1. Enter the Subject.
  2. Enter the Due Date.
  3. Enter a Message.
  4. Click or tap Request to send the request.

OR

  1. Click or tap Cancel to cancel the request.

FIGURE 1-121: Request Updates

Responding To A Request

Users on the receiving end of an update request will received a notification, as seen in FIGURE 1-122:

  1. Click or tap the Input Required button.

FIGURE 1-122: Notification Bar

FIGURE 1-123 shows the input area for the notification:

  1. Enter data as necessary.
  2. If additional entities are available for the user to edit, use the arrows to cycle between entities.
  3. Click or tap Dismiss to cancel changes.
  4. Click or tap Submit to save changes.

FIGURE 1-123: Input Data In Notification

Sending Data To HHS

Users with the appropriate permissions can send HAvBED data to HHS, as shown in FIGURE 1-124:

  1. Click or tap the Send to HHS button.

FIGURE 1-124: Send to HHS

Follow the prompt, as shown in FIGURE 1-121:

  1. Click or tap Transmit to send the data to HHS.

OR

  1. Click or tap Cancel to cancel the transmission .

FIGURE 1-125: Transmit

Viewing Submission History

Users may view previously submitted HHS data transmissions, as shown in FIGURE 1-126:

  1. Click or tap the History tab. Previous submissions and status will show.
  2. Click or tap the View button on any submission to see the raw XML transmission that was sent to HHS.

FIGURE 1-126: History
