Dashboards

Dashboards allow an administrator to set up a shared dashboard for a group of users. These dashboard assignments are managed through Dashboards.

To manage group dashboards, as shown in FIGURE 4-54:

  1. Click or tap on the Users and Permissions icon.
  2. Click or tap on the Dashboards icon under the Manage My Users section.

FIGURE 4-54: Group Dashboard

FIGURE 4-55 shows the main Dashboards page. This page displays all existing dashboards.

FIGURE 4-55: Global Permissions List

Creating a Dashboard

To create a new dashboard , as shown in FIGURE 4-56:

  1. Click or tap the Create New Dashboard button.

FIGURE 4-56: Create New Dashboard Button

The Add Dashboard tool will display with the Details tab already selected, as shown in FIGURE 4-57.

  1. Enter a Name.
  2. Click or tap the Groups tab.

FIGURE 4-57: Add Dashboard

In FIGURE 4-58,assign groups to access the dashboard.

  1. Click or tap the Add Groups button to apply the permission set to a user group. Select user groups from the available groups. For more information, See "Selecting Items from an Available List"

FIGURE 4-58: Create Permission Set - Groups

To exit the global permissions tool, as shown in FIGURE 4-59:

  1. Click or tap the Save button to keep any changes.

OR

  1. Click or tap the Back button to abandon any changes and close the tool.

FIGURE 4-59: Save and Back buttons

Other Permission Set Action(s)

One other action, delete, may be performed with global permission sets.

To perform this action on a global permission set, as shown in FIGURE 4-60:

  1. Click or tap the circle of the desired item.
  2. Click or tap the Actions button.

FIGURE 4-60: Actions Button

Select the desired action and follow the prompt, as shown in FIGURE 4-61.

  1. Delete removes the permission set from the system.

FIGURE 4-61: Permission Set - Other Actions
