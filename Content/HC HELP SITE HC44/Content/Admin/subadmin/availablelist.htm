Selecting Items from an Available List

In different locations in Admin mode, it is possible to select items from a list of all objects of a certain type that are currently available in the system. FIGURE 1-198 shows an example of how a Sub-Admin can assign a user to multiple user groups. Instructions for accessing tools, like the Edit User tool, are provided in their respective sections within this manual.

 

Note: Items are added to or removed from the list the same way in all scenarios.

To add available items to the new list, as shown in FIGURE 1-198:

  1. Click or tap on the Add <Item> button to open the Add <item> dialog box. In FIGURE 1-198, the Add <Item> button is labeled Add Matrices.

FIGURE 1-198: Add Matrices Button

In FIGURE 1-199, user groups are being added or assigned to a user.

  1. Click or tap the circle for each item to be added.
  2. Click or tap the Add Selected button.
  3. Click or tap the Done button to close the Add <Item> dialog box.

FIGURE 1-199: Add Items

In FIGURE 1-200, a user group is being removed from the user. To remove items from a list:

  1. Click or tap the circle for each item to be removed.
  2. Click or tap the Remove Selected button.

FIGURE 1-200: Remove Selected Items
