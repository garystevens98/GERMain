Users and Groups

The Users category contains multiple tools including the Users tool, for management of individual user accounts, and the Groups tool, which allows users to be associated and assigned permissions en masse. FIGURE 1-335 shows the Users category.

FIGURE 1-335: Users and Groups

Access to HC Standard速 workspaces, containing all data in the system, is granted only to user groups that have been assigned to those workspaces and given a set of permissions. The Users and Groups tools are used to provide users with the ability to view and work with data in HC Standard速. A user who is not assigned to any groups will see nothing in HC Standard速.

Users

To manage users, as shown in FIGURE 1-336:

  1. Click or tap on the Users and Permissions icon.
  2. Click or tap on the Users icon under the Manage My Users section.

FIGURE 1-336: Users

On the main Users page, all users under the Sub-Admin's jurisdiction are listed by name, along with preferences and other information associated with each account.

Creating Users

To create a new user in the system, as shown in FIGURE 1-337:

  1. Click or tap the Create New User button.

FIGURE 1-337: Create New User Button

The Add User tool will open with the Profile tab already selected. The following are required fields to enter and are represented with a red asterisk, as shown in FIGURE 1-338:

  1. Enter a Name for the account (usually the full name of the person who will be using it).
  2. Enter a User Name that will be required at login.
  3. Enter a Password and Confirm Password for the user.
  4. Enter a PasswordQuestion and Answer.
  5. Enter an Email address for the user.

FIGURE 1-338: Add User - Profile

No other fields are required to be entered. However, if a phone number is not provided, the user will not receive text messages containing notifications.

FIGURE 1-339 shows additional fields to enter for a user.

  1. Leave the Administrator checkbox unchecked. Sub-Admins may not create Administrator users.
  2. Click or tap the Must Reset Password checkbox to require the user change the password immediately upon logging into the system with the existing password for the first time.
  3. Enter Cell and Pager information to receive notifications via SMS text.
  4. Click or tap the SMS, Email, or Notification Popups checkboxes to receive notifications by the desired method.
  5. Click or tap the Allow Personal Dashboard checkbox to provide the ability to customize dashboards in the HC Standard速 based on accessibility of information. Due to the amount of bandwidth used by multiple dashboards, there is a limit of six items that can be added. Exceptions can be made and access to unlimited dashboard items can be granted by clicking or tapping the Allow Unlimited Dashboard Items checkbox.
  6. Click or tap the Stay logged in even when not performing any activities checkbox to allow the user to stay logged in even when the user has not performed any actions in a certain amount of time.

FIGURE 1-339: User Profile - Additional Fields

Notification settings can be entered when a new account is created in Admin mode, but it can also be entered by the user when logged into HC Standard速 using the User Profile.

The Groups tab of the tool allows the Sub-Admin to assign a user to a user group or multiple groups. Only groups that the Sub-Admin has jurisdiction over may be assigned to a user.

To assign a user to a user group(s), as shown in FIGURE 1-340:

  1. Click or tap the Groups tab.
  2. Click or tap the Add Group button. Select from the available groups, See "Selecting Items from an Available List" These assignments can be made at a later time, if desired, or if the appropriate groups have not yet been created.

FIGURE 1-340: Add User - Groups

When done adding information to the profile of the Add User tool, exit the tool, as shown in FIGURE 1-341:

  1. Click or tap the Save button to keep any changes.

OR

  1. Click or tap the Back button to abandon any changes and close the tool.

FIGURE 1-341: Save and Back Buttons

Other User Action(s)

Other actions may be performed with users including deleting an account, resetting a password, and unlocking an account.

To perform any of the above actions on one or more users, as shown in FIGURE 1-342:

  1. Click or tap the circle of the desired item.
  2. Click or tap the Actions button.

FIGURE 1-342: Users - Actions Button

Select the desired action and follow the prompt, as shown in FIGURE 1-343:

  1. Delete removes the account from the system.
  2. Reset Password allows for a new password to be assigned to the account. The current password is not required to perform this action.
  3. Unlock Account allows for the unlocking of an account that has been locked out due to multiple failed login attempts.

FIGURE 1-343: User Actions
