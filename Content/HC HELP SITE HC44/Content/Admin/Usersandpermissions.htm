Users and Permissions

 

FIGURE 1-201: Users, User Groups, and Permissions

The Users and Permissions category contains the Users tool for management of individual user accounts, and the Groups tool, which allows users to be associated and assigned permissions en masse.

Access to HC Standard速 is granted only to those who have a user account with a valid username and password. Access to HC Standard速 workspaces containing all data in the system is granted only to user groups that have been assigned to those Workspaces and given a set of permissions. To allow someone to view and work with data in HC Standard速, you will use the Users, Groups and Permission tools. For example, a user that is not in any groups will see nothing in HC Standard速.
